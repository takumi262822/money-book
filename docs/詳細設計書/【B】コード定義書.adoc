= MoneyBook — コード定義書
:toc: left
:toclevels: 2

== 概要

本書はソースコード中で使用される識別子（クラス名・関数名・変数名・LocalStorage キー）を定義する。

== クラス一覧

[cols="2,3"]
|===
| クラス名 | 説明

| `KakeiboEngine` | アプリ全体の初期化・イベント委譲・画面更新を統括するメインクラス
| `BudgetService` | 収支レコードの CRUD 操作と月次集計を担うロジッククラス
| `StorageManager` | LocalStorage への読み書きを一元管理するクラス
| `StateManager` | 表示月とモーダル表示状態を保持するクラス
| `ChartManager` | Chart.js を使った支出カテゴリチャートのラッパークラス
| `CsvManager` | 収支レコードを CSV 形式に変換してダウンロードさせるクラス
|===

== 主要関数一覧

[cols="2,2,3"]
|===
| 関数名 | 所属ファイル | 説明

| `renderHeader(yearMonth)` | `src/ui/header.js` | ヘッダー HTML 文字列を生成して返す
| `renderFooter()` | `src/ui/footer.js` | フッター HTML 文字列を生成して返す
| `renderRecordItem(record)` | `src/ui/components.js` | 収支レコード 1 件分のカード HTML を生成して返す
| `renderCategoryOptions(type, selected)` | `src/ui/components.js` | 種別に応じたカテゴリ選択肢 HTML を生成して返す
| `validateRecord(data)` | `src/utils/validator.js` | レコード入力を検証してエラーコード配列を返す
| `validateBudget(data)` | `src/utils/validator.js` | 予算設定入力を検証してエラーコード配列を返す
| `escapeHtml(str)` | `src/utils/xss.js` | HTML 特殊文字をエスケープして XSS を防ぐ
| `formatCurrency(amount)` | `src/utils/formatter.js` | 数値を `¥1,000` 形式にフォーマットする
| `formatDateShort(dateStr)` | `src/utils/formatter.js` | `YYYY-MM-DD` を `M/D` に変換する
| `formatYearMonth(yearMonth)` | `src/utils/formatter.js` | `YYYY-MM` を `YYYY年M月` に変換する
| `getCurrentYearMonth()` | `src/utils/formatter.js` | 現在の年月を `YYYY-MM` 形式で返す
|===

== LocalStorage キー

[cols="2,1,3"]
|===
| キー名 | 定義定数 | 格納内容

| `moneybook_records` | `STORAGE_KEYS.RECORDS` | 収支レコードの JSON 配列
| `moneybook_budgets` | `STORAGE_KEYS.BUDGETS` | 予算設定の JSON オブジェクト
|===
