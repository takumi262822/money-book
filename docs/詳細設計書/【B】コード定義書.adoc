== 概要

本書はプロジェクト「{project}」に登場するコード値（識別子・種別値）を列挙したものです。
ソースファイル src/utils/constants.js から逆引きで抽出しています。

---

== エラーコード（ERROR_CODES）

ファイル: src/utils/constants.js

[cols="1,3,3,2,1,4,3", options="header"]
|===
|項番
|コード和名
|コード英名
|データ型
|桁数
|説明
|記述形式

|1
|金額不正
|INVALID_AMOUNT
|文字型
|14
|金額が正の整数でない、または未入力
|半角英大文字・アンダースコア区切り

|2
|日付不正
|INVALID_DATE
|文字型
|12
|日付が YYYY-MM-DD 形式でない、または未入力
|半角英大文字・アンダースコア区切り

|3
|カテゴリ未選択
|MISSING_CATEGORY
|文字型
|16
|カテゴリが候補リストに含まれない、または未選択
|半角英大文字・アンダースコア区切り

|4
|種別不正
|INVALID_TYPE
|文字型
|12
|種別が income / expense 以外の値
|半角英大文字・アンダースコア区切り
